## Jun Rui trading charts
This project is trading charts demo for jun rui org

### Set Up

```bash
$ (c)npm install
```

### Run Dev

* Run server side data feeds first it will
data feeds under
[localhost:3000](localhost:3000)
```bash
$ npm run datafeeds
```

* run vue client
```bash
$ npm run serve
```
